Welcome to Airport monitor sample app !

It's a SPA (Single Page Application) using AngularJS and REST.
Developed in Java 1.8 with use of IDEA IntelliJ 15. More about technologies below.
Application is developed to test and learn Spring Websockets API. Currently it has no database only mocks.
Frontend :
  • HTML5
  • CSS
  • AngularJS
  • STOCK.js
Backend :
  • Java 1.8
  • Lombok 1.16.4
  • JUnit 4.12
Build tools :
  • IDEA IntelliJ 15
  • Maven 3
  • Spring Boot
  • GIT
Spring Framework :
  • Spring Boot 1.2.7
  • Core
  • Test Framework
  • Web Sockets
  • REST
